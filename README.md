# MQTT 环境监控
简要说明：基于 MQTT over WebSocket 的前端监控页面，用于实时展示温度、湿度、风速和光照等环境数据。

## 📚 开发日志


## V5.0.3 更新内容（2025-12-30）

**图表交互增强**:
- 📊 增加图表互动控件：可暂停实时更新、切换历史视图、重置缩放；图表支持 `dataZoom` 缩放与拖拽查看历史数据。

**MQTT 连接策略调整**:
- 🔁 将 MQTT 重连严格限制为最多 3 次（包含首次连接），超过后显示提示并停止重连，避免无限重试造成误导。
